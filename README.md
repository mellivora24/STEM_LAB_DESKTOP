# STEM_LAB_DESKTOP

Hãy đảm bảo rằng đã cài đặt các thư viện bằng:

~~~bash
pip install -r requirements.txt
~~~

Và active môi trường python bằng các lệnh CMD
~~~bash
python -m venv .venv
~~~
~~~bash
source .venv/bin/activate
~~~
